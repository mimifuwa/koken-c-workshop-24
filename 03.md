## 繰り返しで遊んでみる

C で使うことができる繰り返しは、だいたい 2 つあります。

`for`と`while`です。

こんなふうに使います。

```c
for(int i = 0; i < 33; i++){
  // 繰り返したい処理
}

while(条件){
  // 繰り返したい処理
}
```

繰り返しは、`break`で抜けることができます。
`i++`は、`i = i + 1`ってことです。`i += 1`とも書けます。

---

## 具体的に例を見てみたい

1 から 10 まで表示するプログラム

```c
#include <stdio.h>

int main(void){
  for(int i = 1; i =< 10; i++){
    printf("%d\n", i);
  }

  int i = 1;
  while(i <= 10){
    printf("%d\n", i);
    i++;
  }
}
```

## 練習問題

---

## 答え合わせたいむ。

```c
for(int i = 0; i <= 20; i++){
  printf("%d\n", i);
}

int i = 0;
while(i <= 20){
  printf("%d\n", i);
  i++;
}
```

---

ちなみに逆順

```c
for(int i = 20; i >= 0; i--){
  printf("%d\n", i);
}

int i = 20;
while(i >= 0){
  printf("%d\n", i);
  i--;
}
```
